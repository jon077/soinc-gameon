Science Olympiad: Game On

Scratch skills for "Science Olympiad: Game On"

About These Tutorials

Learn to code Scratch and understand what elements judges will be looking for during the Science Olympiad competition.

The tutorials are listed below and build upon each other creating a fun game.

The star of the show is "Slice" a fun loving pizza slice that is your typical middle-school kid. His favorite foods are chicken tenders, mozzerella sticks and lots of ketchup.

Prerequisites

An open mind, lots of time, and a promise to keep trying.

  • Go to https://www.soinc.org/game-b and watch the videos and understand the scoring rubric and scoring rubric explaination.
  • Go to https://scratch.mit.edu/ and create a Scratch account.

Tutorials

1. Introduction and Help/Instructions

Learn how to create a sprite, an intro screen and help menu covering the rubric requirements. Stages, backdrops and sprites are covered. Simple vector based drawing is also introduced.

2. User Controlled (UC) Sprite

We have an interactive discussion about how to reformulate a real and subtly complicated business problem as a formal machine learning problem. The real goal isn't so much to solve the problem, as to convey the point that properly mapping your business problem to a machine learning problem is both extremely important and often quite challenging. This course doesn't dwell on how to do this mapping, though see Provost and Fawcett's book in the references.

Assignments

Homework 1

Introduction and Help/Instructions

Create a game that contains a introduction, help, and instruction stage.

  • Create an introduction backdop.
  • Create a help backdrop.
  • Create a button to show the instructions and go back to the introduction

Homework 2

User Controlled (UC) Sprite

Create a main character and add smooth movements

  • Create a character using paint tool (vector based)
  • Add smooth movement to the character
  • Flip the character
  • Return the character to an original location
